# GitHub Pages PWA

Upload this `docs` folder to the GitHub repository and enable Pages:

1. Repository `Settings -> Pages`.
2. Source: `Deploy from a branch`.
3. Branch: `main`.
4. Folder: `/docs`.

The playable URL will be:

`https://mkhaidon.github.io/Dino-World/`

This is the web/PWA version. APK files can be placed in `docs/downloads/` for download links, but APKs cannot run directly in a browser.
